# WebGL 3D 模型编辑器

> 本项目由 [www.autoa2a.org](https://www.autoa2a.org) 「AI 研究院」**多个 AI 协作自动生成**（共 5 个页面）。在线访问： https://AutoA2A.github.io/autoatoa-webgl-3d-editor-15-4903/

## 网站简介
本项目是一个基于WebGL的3D模型编辑器网站，旨在为用户提供一个在线编辑和预览3D模型的环境。网站采用多AI协作的方式进行开发，利用不同的AI模型完成不同的任务，如页面设计、功能实现、内容生成等。

## 页面与功能
网站包含以下5个页面：
1. 首页：介绍网站的功能和特点，提供导航链接到其他页面。
2. 模型编辑器：主要功能页面，提供3D模型编辑和预览功能。
3. 模型库：展示可用的3D模型，并提供下载和导入到编辑器的功能。
4. 帮助文档：提供使用网站和编辑器的指南和教程。
5. 关于我们：介绍网站开发团队和相关信息。

主要功能包括：
- 3D模型导入和导出
- 模型缩放、旋转和平移
- 基本的模型编辑操作，如拉伸、剪切等
- 预览和渲染3D模型

## 多 AI 协作与验收
本项目采用多AI协作的方式进行开发，不同的AI模型负责不同的任务：
- 页面设计AI：负责设计网站的页面布局和UI。
- 功能实现AI：负责实现网站的各项功能，如3D模型编辑和预览。
- 内容生成AI：负责生成网站的内容，如帮助文档和关于我们页面。

为了确保项目的质量和进度，我们制定了严格的验收标准：
- 页面设计符合预期，UI友好且易于使用。
- 功能实现完整且正确，能够满足用户需求。
- 内容生成准确且完整，能够为用户提供有用的信息。

## GitHub Pages 部署与访问
本项目已部署到GitHub Pages，可以通过以下入口访问：
- `index.html`：网站的入口页面，位于项目的根目录。

访问方式：
1. 打开GitHub仓库页面。
2. 点击`Settings`按钮。
3. 在`GitHub Pages`部分找到发布的网站链接。
4. 点击链接访问网站。

或者直接访问：`https://<用户名>.github.io/<仓库名>/index.html`

替换`<用户名>`和`<仓库名>`为实际的GitHub用户名和仓库名。
